Portfolio

 

Mid Term Project

For this project we created a Library App that allows users to check in an item and check out an item, as well as search for titles by author/title. We used a stream reader to save all the data to a text file so that the status of books can be saved for the next time the program starts. My co-developers were Kyle G., Shamita, and Reggie.

Movie Category Database

For this project I created a movie category database. The user is able to search for movies based on their categories. 

Pig Latin Translator

For this project I created a translation app that allows the user to enter a phrase and see what it would be in pig latin. I used Lists to store the strings and iterated through them to find where to put the right words.
